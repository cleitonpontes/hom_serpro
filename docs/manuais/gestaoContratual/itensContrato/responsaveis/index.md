# Contratos - Itens Contrato – Responsáveis

## 1. Pesquisa para Inclusão de Responsáveis

Para pesquisar o contrato e incluir um responsável, clique no menu:

Gestão Contratual >> Contratos

No campo Pesquisar, informe os dados do contrato desejado.

![FIGURA 1 - Pesquisa de Contratos](./images/figura1.JPG)

Será apresentada uma tela com resultado da pesquisa.

Clique no ícone “![configurar](../../../icons/configurar.JPG)“ e, na lista de itens Contrato, selecione
“Responsáveis”.

![FIGURA 2 - Seleção de Itens de Contrato](./images/figura2.JPG)


## 2. Adicionar Responsáveis

Para adicionar responsável , clique em “Adicionar Responsável do Contrato”.

![FIGURA 3 - Adicionar Responsável](./images/figura3.JPG)

Preencha os campos dos dados solicitados. Após, clique em
“Salvar e voltar”.

Os campos marcados com “*” são de preenchimento obrigatório.

![FIGURA 4 - Inclusão de Responsável](./images/figura4.JPG)

## 3. Pesquisa de Responsáveis
Para pesquisar o cadastro de um responsável, clique no campo “Pesquisar” e
informe os dados (Tipo Responsável,CPF/CNPJ/UG/ID GÉNÉRICO ou
NOME/RAZÃO SOCIAL).

Na tabela de responsáveis serão apresentados os resultados da pesquisa.

![FIGURA 5 - Pesquisa de Responsáveis](./images/figura5.JPG)

## 4. Editar Responsável

Para editar o cadastro de um responsável, clique no ícone “![editar](../../../icons/editar.JPG)“.

![FIGURA 6 - Lista de Cadastro de Responsável](./images/figura6.JPG)

Será apresentada uma tela com os dados do responsável para edição.

Após a edição, clique em “Salvar e voltar”

![FIGURA 7 - Editar Responsável](./images/figura7.JPG)

## 5. Detalhar Responsável
Para detalhar o cadastro de responsável, clique no ícone “![detalhar](../../../icons/detalhar.JPG)“.

![FIGURA 8 - Detalhar Responsável](./images/figura8.JPG)

Será apresentada uma tela com os detalhes do cadastro do responsável
selecionado.
![FIGURA 9 - Detalhe de Responsável](./images/figura9.JPG)

## 6. Excluir Responsáveis

Para excluir o cadastro de responsável, clique no ícone “![deletar](../../../icons/deletar.JPG)“.

![FIGURA 10 - Exclusão Responsável](./images/figura10.JPG)
